PACMAN

Use arrow keys to move.

This game implements A* algorithm to perform the movement of the ghosts. There're 3 modes of movement:

- Invincible state: the ghost destination is set in Pacman.

- Vulnerable state: the ghost destination is set in a random tile.

- Dead state: the ghost destination is set in the ghosts' spawn.

Some of the extra features that has been implemented are:

- Sound effects: Pack resource

- Differents levels of dificulty: ghosts' velocity is incremented every 30 seconds.

- Score: collect dots rise the score 10 points, super dots give you 50 points and eating a ghost collects 200 points.
